- How to use:
1. Install .NET 7.0.4
2. Install SQL Server 2019
3. Install SQL Server Management Studio
4. Create a server with SQL Server Authentication
5. Set up server login and password
6. Edit the Walmart Context connection string to: "Data Source=localhost,1433; Initial Catalog=walmart; User ID=sa; Password=your_password; TrustServerCertificate=true"
7. Host and run the web using cmd
